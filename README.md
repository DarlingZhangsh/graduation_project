##### 软件架构
本系统是基于SpringSecurity+SpringBoot+Mybatis。 SpringSecurity是一个高度自定义的安全框架。利用Spring IoC/DI和AOP功能，为系统提供了声明式安全访问控制功能，减少了为系统安全而编写大量重复代码的工作
